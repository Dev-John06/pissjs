# PissJS

PissJS is a NPM package for pissing.

## Installation

Use the package manager [npm](https://www.npmjs.com/package/pissjs) to install PissJS.

```bash
npm install pissjs
```

## Usage

```javascript
const PissJS = require("pissjs")

// Console logs piss in a yellow font colour
PissJS.Piss();

```
